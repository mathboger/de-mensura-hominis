# De Mensura Hominis

The gist: a live book that will be publicly written, text and code, on
mathematical modelling aimed at understanding human behaviour, from cell to
society, in order to help create a better future for all of us. More details
on the motivation and content can be found in the book itself. This is also
a project used for studying. I am always open to communication, so send me a
message if interested!

glia@mathboger.nl
Matheus Boger

**Note:** I used AI to build the technical skeleton for this project. I did
check the code to make sure it was doing what I intended. For me, the most
important part of the book is that the words in the actual text reflect my
own thought and that the understanding of the matter can be built by the
reader. This means that I see AI as a tool to help with research and
implementation, but everything that gets published is thoroughly reviewed by
me first. All text output in the HTML and PDF versions is authorial.

## Building locally

### Prerequisites

| Tool | Version | Install |
|------|---------|---------|
| [Quarto](https://quarto.org) | ≥ 1.4 | `brew install quarto` / [installer](https://quarto.org/docs/get-started/) |
| Python | ≥ 3.10 | `brew install python` |
| R | ≥ 4.3 | [CRAN](https://cran.r-project.org) |
| TinyTeX | any | `quarto install tinytex` |

For additional information refer to SETUP.md.
